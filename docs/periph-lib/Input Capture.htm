1          Introduction

The capture peripheral is supported with the following functions:

Table:             Input Capture Functions

Function

Description

CloseCapturex

Disable capture peripheral x.

OpenCapturex

Configure capture peripheral x.

ReadCapturex

Read a value from capture peripheral x.

CloseECapture1

Disable enhanced capture peripheral 1.

OpenECapture1

Configure enhanced capture peripheral 1.

ReadECapture1

Read a value from enhanced capture peripheral 1.

 

 

All PIC18 devices are divided into following different CC versions. Before calling the LIB functions care has to be taken to identify the version of the configured device and to call the appropriate function with correct number of arguments. For CC_V7 user has to configure the I/O registers.

2.2         OpenCapture1
OpenCapture2
OpenCapture3
OpenCapture4
OpenCapture5
OpenECapture1

Function:

Configure and enable input capture x.

Include:

capture.h

Prototype:

For CC_V1, CC_V2, CC_V3, CC_V4, CC_V5 , CC_V6 and CC_V7:

void OpenCapture1( unsigned char config );

For CC_V2, CC_V3, CC_V4 , CC_V6 and CC_V7:

void OpenCapture2( unsigned char config );

For CC_V3 and CC_V4:

void OpenCapture3( unsigned char config );

For CC_V4:

void OpenCapture4( unsigned char config );

void OpenCapture5( unsigned char config );

For ECC_V5:

void OpenECapture1( unsigned char config );

Arguments:

config

A bitmask that is created by performing either bitwise AND operation (‘&’) or bitwise OR operation (‘|’), configurable either way as shown on the example at the end of this file, with a value from each of the categories listed below. These values are defined in the file capture.h:

 

Enable CCP Interrupts:

     CAPTURE_INT_ON                   Interrupts Enabled

     CAPTURE_INT_OFF                 Interrupts Disabled

 

For CC_V1, CC_V2, CC_V3, CC_V4, CC_V5 , CC_V6 and CC_V7:

Capture configuration

     CAP_EVERY_FALL_EDGE         Interrupt on every falling edge

     CAP_EVERY_RISE_EDGE         Interrupt on every rising edge

     CAP_EVERY_4_RISE_EDGE     Interrupt on every 4th rising edge

     CAP_EVERY_16_RISE_EDGE   Interrupt on every 16th rising edge

 

For ECC_V5:

Compare configuration

     ECAP_EVERY_FALL_EDGE       Interrupt on every falling edge (enhanced)

     ECAP_EVERY_RISE_EDGE       Interrupt on every rising edge (enhanced)

     ECAP_EVERY_4_RISE_EDGE   Interrupt on every 4th rising edge
                                                    (enhanced)

     ECAP_EVERY_16_RISE_EDGE Interrupt on every 16th rising edge
                                                    (enhanced)

Remarks:

This function first resets the capture module to the POR state and then configures the input capture for the specified edge detection.

 

NOTE: In addition to opening the capture, the appropriate timer module must be enabled before any of the captures will operate. See the data sheet for CCP and timer interconnect configurations and Timer Functions for the arguments used with CCP in OpenTimer3.

File Name:

cp1open.c

cp2open.c

cp3open.c

cp4open.c

cp5open.c

ep1open.c

Code Example:

With AND mask:

OpenCapture1( CAPTURE_INT_ON &

              CAP_EVERY_4_RISE_EDGE );

 

With OR mask:

OpenCapture1( CAPTURE_INT_ON |

              CAP_EVERY_4_RISE_EDGE );

Function:

Read the result of a capture event from the specified input capture.

Include:

capture.h

Prototype:

For CC_V1, CC_V2, CC_V3, CC_V4, CC_V5, CC_V6  and CC_V7:

unsigned int ReadCapture1( void );

For CC_V2, CC_V3, CC_V4, CC_V6 and CC_V7:

unsigned int ReadCapture2( void );

For CC_V3 and CC_V4:

unsigned int ReadCapture3( void );

For CC_V4:

unsigned int ReadCapture4( void );

unsigned int ReadCapture5( void );

For ECC_V5:

unsigned int ReadECapture1( void );

Remarks:

This function reads the value of the respective input capture CCPRx registers.

Return Value:

This function returns the result of the capture event.

File Name:

cp1read.c

cp2read.c

cp3read.c

cp4read.c

cp5read.c

ep1read.c

 

Example Use of the Capture Routines with AND mask

#include <p18cxxx.h>

#include <capture.h>

#include <timers.h>

 

void main(void)

{

  unsigned int result;

 

  // Configure Capture1

  OpenCapture1( CAP_EVERY_4_RISE_EDGE &

                CAPTURE_INT_ON );

 

  // Configure either Timer1 or Timer3 to the CCP1 module

  OpenTimerx (argument);

 

  while(!PIR1bits.CCP1IF);  // Wait for event

 

  result = ReadCapture1(); // read result

 

  // Clean up

  CloseCapture1();

  CloseTimerx();

  }

 

 

Example Use of the Capture Routines with OR mask

#include <p18cxxx.h>

#define USE_OR_MASKS

#include <capture.h>

#include <timers.h>

 

void main(void)

{

  unsigned int result;

 

  // Configure Capture1

  OpenCapture1( CAP_EVERY_4_RISE_EDGE |

                CAPTURE_INT_ON );

 

  // Configure either Timer1 or Timer3 to the CCP1 module

  OpenTimerx (argument);

 

  while(!PIR1bits.CCP1IF);  // Wait for event

 

  result = ReadCapture1(); // read result

 

  // Clean up

  CloseCapture1();

  CloseTimerx();

  }
