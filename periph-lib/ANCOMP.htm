2.2         Open_ancomp 

2.2.1        Open_ancomp 

For ANCOM_V1, ANCOM_V2, ANCOM_V3 and ANCOM_V4

Function:

Configure the Analog comparator.

Include:

ancomp.h

Prototype:

Open_ancomp( unsigned char config );

Arguments:

Config

A bitmask that is created by performing either bitwise AND operation (‘&’) or bitwise OR operation (‘|’), user configurable, with a value from each of the categories listed below. These values are defined in the file ancomp.h.

 

Comparator Output Inversion:

COMP_1_2_OP_INV               Comparator 1, 2  with OP invert

COMP_1_OP_INV                   Comparator 1  with OP invert

COMP_2_OP_INV                   Comparator 2  with OP invert

COMP_OP_INV_NONE             Comparator 1,2  without OP invert

 

Comparator Mode Select:

COMP_1_2_INDP              Two independent comparator

COMP_1_2_INDP_OP             Two independent comparator with Outputs

COMP_1_2_COMN_REF           Two common ref comparator

COMP_1_2_COMN_REF_OP   Two common ref comparator with Outputs

COMP_1_INDP_OP                 One independent comparator with Output

COMP_INT_REF_SAME_IP  Comparators with int ref and default Inputs

COMP_INT_REF_MUX_IP       Comparators with int ref and cross Inputs

 

Comparator Interrupts:

COMP_INT_EN                          Comparator interrupt enable

COMP_INT_DIS                        Comparator interrupt disable

Remarks:

This function resets the Analog comparator peripheral to the POR state and configures the Analog comparator-related Special Function Registers (SFRs) according to the options specified, also configures the inputs and outputs for the specified options.

File Name:

open_ancomp.c

Code Example:

With AND mask:

Open_ancomp(COMP_1_2_OP_INV   &

                          COMP_1_2_INDP   &

            COMP_INT_EN );

With OR mask:

Open_ancomp(COMP_1_2_OP_INV   |

                          COMP_1_2_INDP   |

            COMP_INT_EN );

 

2.2.2        Open_ancomp1 

For ANCOM_V5, ANCOM_V6, ANCOM_V7, ANCOM_V8 and ANCOM_9

Function:

Configure the Analog comparator1.

Include:

ancomp.h

Prototype:

Open_ancomp1( unsigned char config );

Arguments:

Config

A bitmask that is created by performing either bitwise AND operation (‘&’) or bitwise OR operation (‘|’), configurable either way as shown in the example at the end of this file, with a value from each of the categories listed below. These values are defined in the file ancomp.h.

 

Comparator1 Interrupt:           

COMP_INT_EN             Enables comparator1 Interrupt

COMP_INT_DIS            Disable comparator1 Interrupt

 

Comparator1 Output enable:

COMP_OP_EN                 C1OUT is present on C1OUT pin

COMP_OP_DIS               C1OUT is Internal only

 

Comparator1 output polarity select:

COMP_OP_INV              Comparator1 output logic is Inverted

COMP_OP_NINV            Comparator1 output logic is not Inverted

For ANCOM_V5:

 

Comparator1 Ref (C1VREF )select:

COMP_REF_FVR            C1VREF input connects to FVR (fixed 1.2V)         

COMP_REF_CVREF        C1VREF input connects to CVREF

 

Comparator1 Speed/Power select:

COMP_HSPEED             Comparator1 operates at normal power higher 

                                     speed mode

COMP_LSPEED             Comparator1 operates at low power low  speed mode

 

 

Comparator1 Ref (C1VIN+) select:

COMP_VINP_PIN            C1VIN+ connects to C1VIN+ pin         

COMP_VINP_VREF          C1VIN+ connects to C1VREF output

 

Comparator1 channel select:

COMP_VINM_IN0            C12IN0- pin connects to C1VIN-

COMP_VINM_IN1            C12IN1- pin connects to C1VIN-

COMP_VINM_IN2            C12IN2- pin connects to C1VIN-

COMP_VINM_IN3            C12IN3- pin connects to C1VIN-

For ANCOM_V6 and ANCOM_V7:

 

Comparator1 Interrupt polarity select:

COMP_INT_ALL_EDGE        Interrupt on any change of the output

COMP_INT_FALL_EDGE      Interrupt on high-to-low transition of the output

COMP_INT_RISE_EDGE      Interrupt on low-to-high transition of the output 

COMP_INT_NOGEN              Interrupt generation is disabled

 

 

 

Comparator1 Ref (C1VIN+)select:        

COMP_REF_CVREF                 C1VIN+ connects to Internal CVREF         

COMP_REF_CINA                   C1VIN+ connects to C1INA pin    

 

Comparator1 channel select:

COMP_VINM_VIRV        C1VIN- connects to VIRV (fixed Internal Ref.

                                                1.2V )              

COMP_VINM_CIND                 C1VIN- connects to C1IND                     

COMP_VINM_CINC                 C1VIN- connects to C1INC                    

COMP_VINM_CINB            C1VIN- connects to C1INB        

For ANCOM_V8:

Comparator1 Ref (C1VREF )select:

COMP_REF_FVR         C1VREF input connects to FVR          

COMP_REF_DAC                  C1VREF input connects to DAC

 

Comparator1 Speed/Power select:

COMP_HSPEED            Comparator1 operates at normal power higher 

                                     speed mode

COMP_LSPEED             Comparator1 operates at low power low  speed mode

 

Comparator1 Ref (C1VP) select:

COMP_CxVP_VREF    C1VP connects to C1VREF output 

COMP_CxVP_PIN           C1VP connects to RC0 pin

 

Comparator1 channel select:

COMP_VINM_GND     C1VN connects to GND         

COMP_VINM_IN1     C1VN connects to RC1 pin

COMP_VINM_IN2     C1VN connects to RC2 pin     

COMP_VINM_IN3          C1VN connects to RC3 pin

For ANCOM_V9:

Comparator1 Interrupt polarity select:

COMP_INT_ALL_EDGE        Interrupt on any change of the output

COMP_INT_FALL_EDGE      Interrupt on high-to-low transition of the output

COMP_INT_RISE_EDGE      Interrupt on low-to-high transition of the output 

COMP_INT_NOGEN              Interrupt generation is disabled

 

Comparator1 Ref (C1VIN+)select:        

COMP_REF_CVREF                 C1VIN+ connects to Internal CVREF         

COMP_REF_CINA                   C1VIN+ connects to C1INA pin    

 

Comparator1 channel select:

COMP_VINM_VBG         Comparator1 connects to VBG/2

COMP_VINM_CM1_C2INB   Comparator1 connects to C2INB

COMP_VINM_GND                   Comparator(1or2)connects to ground

COMP_VINM_CINB            Comparator connects to C1INB

Remarks:

This function resets the Analog comparator1 peripheral to the POR state and configures the Analog comparator-related Special Function Registers (SFRs) according to the options specified, also configures the inputs and outputs for the specified options.

File Name:

open_ancomp.c

Code Example:

With AND mask:

Open_ancomp1(COMP_OP_NINV           &

             COMP_OP_EN               &

             COMP_INT_ALL_EDGE &

                            COMP_REF_CVREF       &

                            COMP_VINM_CIND    &

             COMP_INT_EN );

With OR mask:

Open_ancomp1(COMP_OP_NINV           |

             COMP_OP_EN                |

             COMP_INT_ALL_EDGE |

                            COMP_REF_CVREF        |

                            COMP_VINM_CIND   |

             COMP_INT_EN );

Example Use of the Analog comparator Routines with AND mask:

#include <p18cxxx.h>

#include <ancomp.h>

 

 

void main( void )

{

  /* configure Analog comparator1 for non inverted output to appear externally(on pin), interrupt on any change of output, noninverting input connected to CVREF, inverting input connected to Input D of the comparator and enable comparator1 peripheral interrupt.  */

Open_ancomp1(COMP_OP_EN           &

             COMP_OP_NINV               &

             COMP_INT_ALL_EDGE  &

                            COMP_REF_CVREF        &

                            COMP_VINM_CIND    &

             COMP_INT_EN );

/* configure Analog comparator2 for internal output with inversion, interrupt on high–to-low transition of output, noninverting input connected to Input A and inverting input connected to Input B of the comparator and enable comparator2 peripheral interrupt.*/  

Open_ancomp2(COMP_OP_DIS               &

             COMP_OP_INV               &

             COMP_INT_FALL_EDGE  &

                            COMP_REF_ CINA         &

                            COMP_VINM_CINB    &

             COMP_INT_EN );

  

close_ancomp1( void );        //Disable comparator1 and associated Interrupt generation mechanism.

close_ancomp2( void );        //Disable comparator2 and associated

Interrupt generation mechanism.

 

}
