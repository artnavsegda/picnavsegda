1          Introduction

PORTB and INTx are supported with the following functions:

Table:             I/O Port Functions

Function

Description

ClosePORTB

Disable the interrupts and internal pull-up resistors for PORTB.

CloseRBxINT

Disable interrupts for INT pin x .

OpenPORTB

Configure the interrupts and internal pull-up resistors on PORTB.

OpenRBxINT

Enable interrupts, pull-up resisters and selects interrupt edge for INT pin x .

 

All PIC18 devices are divided into following different PTB versions. Before calling the LIB functions care has to be taken to identify the version of the configured device and to call the appropriate function with correct number of arguments. For PTB_V6 user has to configure the I/O registers.

2          Function Descriptions

2.1         ClosePORTB

For PTB_V1, PTB_V2, PTB_V3, PTB_V4, PTB_V5 and PTB_V6

Function:

Disable the interrupts and internal pull-up resistors for PORTB.

Include:

portb.h

Prototype:

void ClosePORTB( void );

Remarks:

This function disables the PORTB interrupt-on-change and the internal pull-up resistors.

File Name:

portb.h

 

2.2         CloseRBxINT

Function:

Disable the interrupts for the specified INTx pin.

Include:

portb.h

Prototype:

For PTB_V1, PTB_V2, PTB_V3, PTB_V4, PTB_V5 and PTB_V6

void CloseRB0INT( void );

void CloseRB1INT( void );

void CloseRB2INT( void );

For PTB_V2, PTB_V3 and PTB_V6

void CloseRB3INT( void );

Remarks:

This function disables the INTx interrupt.

File Name:

portb.h

 

2.3         OpenPORTB

For PTB_V1, PTB_V2, PTB_V3, PTB_V4, PTB_V5 and PTB_V6

Function:

Configure the interrupts and internal pull-up resistors on PORTB.

Include:

portb.h

Prototype:

void OpenPORTB( unsigned char config);

Arguments:

Config

A bitmask that is created by performing either bitwise AND operation (‘&’) or bitwise OR operation (‘|’), user configurable, with a value from each of the categories listed below. These values are defined in the file portb.h.

Interrupt-on-change:

      PORTB_CHANGE_INT_ON                Interrupt enabled

      PORTB_CHANGE_INT_OFF  Interrupt disabled

Enable Pullups:

      PORTB_PULLUPS_ON          pull-up resistors enabled

      PORTB_PULLUPS_OFF        pull-up resistors disabled

Remarks:

This function configures the interrupts and internal pull-up resistors on PORTB.

File Name:

pbopen.c

Code Example:

With AND mask:

OpenPORTB( PORTB_CHANGE_INT_ON & PORTB_PULLUPS_ON);

 

With OR mask:

OpenPORTB( PORTB_CHANGE_INT_ON | PORTB_PULLUPS_ON);

 

2.4         OpenRBxINT

Function:

Enable interrupts for the specified INTx pin.

Include:

portb.h

Prototype:

For PTB_V1, PTB_V2, PTB_V3, PTB_V4, PTB_V5 and PTB_V6

void OpenRB0INT( unsigned char config );

void OpenRB1INT( unsigned char config );

void OpenRB2INT( unsigned char config );

For PTB_V2, PTB_V3 and PTB_V6

void OpenRB3INT( unsigned char config );

Arguments:

Config

A bitmask that is created by performing either bitwise AND operation (‘&’) or bitwise OR operation (‘|’), user configurable, with a value from each of the categories listed below. These values are defined in the file portb.h.

Interrupt-on-change:

      PORTB_CHANGE_INT_ON         Interrupt enabled

      PORTB_CHANGE_INT_OFF       Interrupt disabled

Interrupt-on-edge:

      RISING_EDGE_INT           Interrupt on rising edge

      FALLING_EDGE_INT         Interrupt on falling edge

Enable Pullups:

      PORTB_PULLUPS_ON         pull-up resistors enabled

      PORTB_PULLUPS_OFF       pull-up resistors disabled

Remarks:

This function configures the interrupts and internal pull-up resistors on INTx.

File Name:

rb0open.c

rb1open.c

rb2open.c

rb3open.c

Code Example:

With AND mask:

OpenRB0INT( PORTB_CHANGE_INT_ON &

            RISING_EDGE_INT     &

            PORTB_PULLUPS_ON);

 

With OR mask:

OpenRB0INT( PORTB_CHANGE_INT_ON |

            RISING_EDGE_INT     |

            PORTB_PULLUPS_ON);

 

2.5         Macro definitions

Macros

Description

DisablePullups

Disable the internal pull-up resistors on PORTB.

EnablePullups

Enable the internal pull-up resistors on PORTB.

 

 

 

Example Use of the IO Port Routines with AND mask:

#include <p18cxxx.h>

#include <portb.h>

 

void main( void )

{

// configure INT1 with pullups enabled, falling edge and 

// interrupt disable

      OpenRB1INT( PORTB_CHANGE_INT_OFF &

            FALLING_EDGE_INT           &

            PORTB_PULLUPS_ON);

 

      While(INTCON3bits.INT1IF == 0);

      INTCON3bits.INT1IF == 0;

      CloseRB1INT();          //disable INT1

 

}

Example Use of the IO Port Routines with OR mask:

#include <p18cxxx.h>

#define  USE_OR_MASKS

#include <portb.h>

 

void main( void )

{

// configure INT1 with pullups enabled, falling edge and 

// interrupt disable

      OpenRB1INT( PORTB_CHANGE_INT_OFF |

            FALLING_EDGE_INT           |

            PORTB_PULLUPS_ON);

 

      While(INTCON3bits.INT1IF == 0);

      INTCON3bits.INT1IF == 0;

      CloseRB1INT();          //disable INT1

 

}
