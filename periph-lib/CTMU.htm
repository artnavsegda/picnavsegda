1          Introduction

The CTMU peripheral is supported with the following functions:

Table:  CTMU  Functions

Function

Description

OpenCTMU

This function configures the CTMU. 

CurrentControlCTMU   

This function selects the current source range and trims the current source of CTMU. 

CloseCTMU

This function turns off the CTMU module and disables the CTMU interrupts.

 

 

Based on the different control registers, configuration bits and their positions in the control register, all PIC18 devices are divided into the following different versions. Wherever required, separate functions have been designed to support these versions, so before calling the LIB functions care has to be taken to know the version of the configured device and to call the appropriate function with correct number of arguments.

 

Below is the table to find the CTMU version for the configured device:

Prototype:

Arguments:

This contains the parameters to be configured in the CTMUCONH register

 

CTMU Enable/Disable:                       

CTMU_ENABLE                 CTMU is enabled

CTMU_DISABLE                CTMU is disabled

 

 

Idle Mode bit:

CTMU_IDLE_STOP           CTMU discontinue module operation in Idle mode

CTMU_IDLE_CONTINUE    CTMU Operate in Idle mode

Time Generation Enable bit:                            

CTMU_TIME_GEN_ENABLE     CTMU enables edge delay generation

CTMU_TIME_GEN_DISABLE    CTMU disables edge delay generation

Edge Enable bit:

CTMU_EDGE_ENABLE         CTMU edges are not blocked

CTMU_EDGE_DISABLE        CTMU edges are blocked

Edge sequence Enable bit:

CTMU_EDGE_SEQUENCE_ON     Edge1 event must occur before edge2 event                                                  Can occur

CTMU_EDGE_SEQUENCE_OFF   No edge sequence is needed

Analog Current Source Control bit:

CTMU_AN_CURR_SOURCE_GND  CTMU Analog current source output

                                                           is grounded

CTMU_AN_CURR_SOURCE_NOT_GND     CTMU Analog current source                                                   output is not grounded

Trigger Control bit:

CTMU_TRIG_OUTPUT_ENABLE        Trigger output is enabled

CTMU_TRIG_OUTPUT_DISABLE       Trigger output is disabled

Enable/Disable  Interrupt: 

CTMU_INT_ON                                 Enable CTMU interrupts

CTMU_INT_OFF                                Disable CTMU interrupts

2.3           CloseCTMU

Function:

This function turns off the CTMU module and disables the CTMU interrupts.

 

Include:

ctmu.h

 

Prototype:

void CloseCTMU( void )

 

Return Value:

 

None

Remarks:

This function first disables the CTMU interrupt and then turns off the CTMU module. The Interrupt Flag bit is also cleared.

 

File Name:

CloseCTMU.c

 

 

 

3            Macro definitions

Macros

Description

CTMU_INT_ENABLE()

Enables CTMU interrupt i.e. sets PEIE and ADIE bits.

CTMU_INT_DISABLE()

Disables the CTMU interrupt.

Enbl_CTMUEdge1

By setting edge1 status bit current source enables

Enbl_CTMUEdge2

By setting edge2 status bit current source enables

Disbl_CTMUEdge1

By clearing edge1 status bit disable the current source

Disbl_CTMUEdge2

By clearing edge2 status bit disable the current source

CTMUEdge1_Status

Returns the status of CTMU edge1

CTMUEdge2_Status

Returns the status of CTMU edge2

CTMU_Clear_Intr_Status_Bit

Macro to Clear CTMU Interrupt Status bit
