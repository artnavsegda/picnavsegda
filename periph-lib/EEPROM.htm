1          Introduction

The following routines are provided for devices with a single USART peripheral:

Table:             EEPROM Functions

Function

Description

Read_b_eep

Read single byte from Internal EEP

Write_b_eep

Write single byte to Internal EEP

Busy_eep

Checks & waits the status of ER bit in EECON1 register

 

 

Based on the different control registers, configuration bits and their positions in the control register, all PIC18 devices are divided into 3 different versions. Wherever required, separate functions have been designed to support these versions, so before calling the LIB functions care has to be taken to know the version of the configured device and to call the appropriate function with correct number of arguments.

2          Function Descriptions

2.1         Read_b_eep

 

Function:

Read single byte from Internal EEP

Include:

eep.h

Prototype:

unsigned char Read_b_eep( unsigned int badd );

Remarks:

Returns a value after reading the location passed as parameter.

Return Value:

This function returns the read value of the location.

File Name:

read_B.c

Code Example:

Temp = Read_b_eep(0x0000);

 

2.2         Write_b_eep

Function:

Write single byte to Internal EEP

Include:

eep.h

Prototype:

void Write_b_eep(unsigned int badd,unsigned char bdata);

Remarks:

This function writes data in to the specified location inside the EEP

File Name:

write_B.c

Code Example:

Write_b_eep (0x0000,0x20);

 

2.3         Busy_eep

Function:

Checks & waits the status of ER bit in EECON1 register

Include:

eep.h

Prototype:

void Busy_eep ( void );

Remarks:

This function waits until Write cycle to the EEPROM is complete.

File Name:

busy_eep.c

Code Example:

while (!Busy_eep ());

 

Example Use of the EEP Routines:

 

#include <p18Cxxx.h>

#include <eep.h>

 

Void main (void)

{

      /* Write single byte to Internal EEP*/

      Write_b_eep (address, data);

 

      /* Checks & waits the status of ER bit in EECON1 register */

      Busy_eep ();

 

      /* Read single byte from Internal EEP*/

      Temp = Read_b_eep (address);

}
