1          Introduction

The compare peripheral is supported with the following functions:

Table:             OUTput COMPARE Functions

Function

Description

CloseComparex

Disable compare peripheral x.

OpenComparex

Configure compare peripheral x.

CloseECapture1 

Disable enhanced compare peripheral 1.

OpenECapture1 

Configure enhanced compare peripheral 1.

 

 

All PIC18 devices are divided into following different CC versions. Before calling the LIB functions care has to be taken to identify the version of the configured device and to call the appropriate function with correct number of arguments. For CC_V7 user has to configure the I/O registers.

2.2       OpenCompare1
OpenCompare2
OpenCompare3
OpenCompare4
OpenCompare5
OpenECompare1

Function:

Configure and enable output compare x.

Include:

compare.h

Prototype:

For CC_V1, CC_V2, CC_V3, CC_V4, CC_V5, CC_V6 and CC_V7:

void OpenCompare1( unsigned char config, unsigned int period );

For CC_V2, CC_V3, CC_V4, CC_V6 and CC_V7:

void OpenCompare2( unsigned char config, unsigned int period );

For CC_V3 and CC_V4:

void OpenCompare3( unsigned char config, unsigned int period );

For CC_V4:

void OpenCompare4( unsigned char config, unsigned int period );

void OpenCompare5( unsigned char config, unsigned int period );

For ECC_V5:

void OpenECompare1( unsigned char config, unsigned int period );

Arguments:

Config

A bitmask that is created by performing either bitwise AND operation (&) or bitwise OR operation (|), configurable either way as shown on the example at the end of this file, with a value from each of the categories listed below. These values are defined in the file compare.h.

 

Enable CCP Interrupts:

     COM_INT_ON                           Interrupt Enabled

     COM_INT_OFF                         Interrupt Disabled

 

For CC_V1, CC_V2, CC_V3, CC_V4, CC_V5, CC_V6 and CC_V7:

Compare configuration

     COM_TOGG_MATCH                   Toggle output on match

     COM_HI_MATCH                       Output HI on match

     COM_LO_MATCH                       Output LO on match

     COM_UNCHG_MATCH                 Output unchanged on match

     COM_TRIG_SEVNT         Trigger Special Event

 

For ECC_V5:

Compare configuration

     ECOM_TOGG_MATCH                 Toggle output on match

     ECOM_HI_MATCH                     Output HI on match

     ECOM_LO_MATCH                     Output LO on match

     ECOM_UNCHG_MATCH               Output unchanged on match

     ECOM_TRIG_SEVNT         Trigger Special Event

 

 

period

This is the 16bit data, to be compared with TMRx values, it can have any value from 1 to 65535.

Remarks:

This function first resets the compare module to the POR state and then configures the output compare for the specified event, also the 16bit period data is assigned to the CCPRx registers.

 

NOTE: In addition to opening the compare, the appropriate timer module must be enabled before any of the compares will operate. See the data sheet for CCP and timer interconnect configurations.

File Name:

cm1open.c

cm2open.c

cm3open.c

cm4open.c

cm5open.c

em1open.c

Code Example:

With AND mask:

OpenCompare1( COM_INT_ON &

              COM_LO_MATCH,0x1234);

With OR mask:

OpenCompare1( COM_INT_ON |

              COM_LO_MATCH,0x1234);

 

 

Example Use of the Compare Routines with AND mask

#include <p18cxxx.h>

#include <compare.h>

#include <timers.h>

 

void main(void)

{

  // Configure Compare1 for ccp1 pin to set high on match and also

     generate interrupt.

  OpenCompare1( COM_INT_ON &

               COM_HI_MATCH,0x1234);

 

  // Configure either Timer1 or Timer3 to the CCP1 module

   OpenTimerx (argument);

 

  CloseCompare1();

  CloseTimerx();

 }

 

Example Use of the Compare Routines with OR mask

#include <p18cxxx.h>

#define  USE_OR_MASKS

#include <timers.h>

 

void main(void)

{

  // Configure Compare1 for ccp1 pin to set high on match and also      

     generate interrupt.

  OpenCompare1( COM_INT_ON |

               COM_HI_MATCH, 0x1234);

 

  // Configure either Timer1 or Timer3 to the CCP1 module

   OpenTimerx (argument);

 

  CloseCompare1();

  CloseTimerx();

}
